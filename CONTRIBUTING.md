# Contributing to geowriter/aiwriter

Thank you for your interest in the official AI writing skill standards for **geowriter.ai**!

To maintain the rigor and consistency of the `aiwriter` core logic, we have established the following contribution guidelines:

## 🔐 Development Policy

This project operates on an **"Open Standard, Closed Development"** model:

1. **Core Development**: Development and write access (Push/Merge) are strictly reserved for the **geowriter.ai** official team and authorized partners.
2. **External Pull Requests**: To ensure the stability of our writing standards, we are **not accepting** external Pull Requests at this time. If you have suggestions for improvement, please communicate with us via [Issues](#-feedback--suggestions).

---

## 📢 Feedback & Suggestions

While direct code contributions are restricted, we highly value community feedback! You can participate in the following ways:

* **Bug Reports**: If you find logical errors or generation anomalies in a Skill definition, please open an Issue with a reproducible prompt case.
* **Feature Proposals**: If you would like `aiwriter` to support more writing scenarios (e.g., specific industry copywriting standards), feel free to describe your requirements in an Issue.
* **Commercial Use**: If you intend to integrate these skills into your own commercial products, please comply with the [Apache License 2.0](./LICENSE) and provide proper attribution.
